# alexsalom.es

[![Netlify Status](https://api.netlify.com/api/v1/badges/b0bac890-d559-4bbb-a3ff-6e98a7e1a8f8/deploy-status)](https://app.netlify.com/projects/alexsalom/deploys)

Personal site built with [Hugo](https://gohugo.io/) and a custom theme. Deployed automatically to [Netlify](https://www.netlify.com/) on every push to `master`.

## Local development

```bash
hugo server
```

Then open http://localhost:1313/.

## Build

```bash
hugo --minify
```

Output goes to `public/`.
